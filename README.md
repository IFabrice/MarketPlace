# Ndangira

> *"Ndangira" вЂ” Kinyarwanda*

A hyper-local marketplace app connecting vendors in major Rwandan markets (Kigali & beyond) with local consumers through a structured, location-based digital storefront.

**Platforms:** iOS & Android (Flutter cross-platform)

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Flutter (Dart) |
| Auth | Firebase Phone OTP |
| Database | Cloud Firestore |
| File Storage | Firebase Storage |
| Messaging | Firebase Cloud Messaging (FCM) |
| Payments | Flutterwave / Paypack (MTN MoMo, Airtel Money) |
| Maps | Google Maps Flutter |
| State | Flutter Riverpod |
| Navigation | GoRouter |

## Firestore Collections

| Collection | Key Fields |
|---|---|
| `Users` | uid, phone, role (client/vendor), name |
| `Markets` | marketID, name, city, prefix, totalStalls |
| `Vendors` | vendorID, marketID (null if individual), stallCode, rating |
| `Products` | productID, vendorID, name, price (RWF), imageURL, category, inStock |
| `Chats` | chatID, participants[], lastMessage, timestamp |

## Getting Started

### 1. Prerequisites

- Flutter SDK в‰Ґ 3.3.0
- Firebase CLI + FlutterFire CLI
- A Google Firebase project

### 2. Clone and install dependencies

```bash
git clone <repo-url>
cd MarketPlace
flutter pub get
```

### 3. Configure Firebase

```bash
# Install FlutterFire CLI (once)
dart pub global activate flutterfire_cli

# Login to Firebase
firebase login

# Generate firebase_options.dart
flutterfire configure
```

Select your Firebase project and enable:
- **Authentication** в†’ Phone provider
- **Cloud Firestore**
- **Firebase Storage**
- **Cloud Messaging**

### 4. Add fonts

Download the [Inter font family](https://fonts.google.com/specimen/Inter) and place the `.ttf` files in `assets/fonts/`:
- `Inter-Regular.ttf`
- `Inter-Medium.ttf`
- `Inter-SemiBold.ttf`
- `Inter-Bold.ttf`

### 5. Seed market data

Call `FirestoreService().seedMarkets()` once on first launch (or via a Firestore admin script) to populate the Markets collection with Kigali market data.

### 6. Run the app

```bash
flutter run
```

## Stall Code Format

Market vendor stall codes follow the pattern: `[PREFIX]-[AISLE][ROW]-[STALL]`

| Example | Market | Aisle | Stall |
|---|---|---|---|
| `NYA-A2-04` | Nyabugogo | A, row 2 | Stall 4 |
| `KIM-B1-12` | Kimironko | B, row 1 | Stall 12 |

The **Grid Locator** decodes these codes into plain-text directions shown to customers.
